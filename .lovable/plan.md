

# Sanjay Krishna K — Portfolio Website

## Design Direction
Dark, techy aesthetic with a deep dark background, neon cyan/green accent colors, terminal-inspired typography, and heavy animations throughout — scroll reveals, typing effects, particle/grid backgrounds, and glowing hover states.

---

## Page Structure (Single-Page Scrolling)

### 1. Hero Section
- Full-screen dark hero with animated particle/grid background
- Typing animation for the tagline ("Building Intelligent, Scalable & Cloud-Native Systems")
- Name, title, and key roles (Agentic AI Engineer | ML Architect | Full Stack Developer | Cloud Engineer)
- Glowing stat counters (20+ Projects, 12 AI Systems, 8 Platforms) that animate on scroll
- Scroll-down indicator with bounce animation

### 2. About Me
- Brief bio with fade-in animations
- Animated pipeline visualization: Data → Intelligence → Automation → Impact
- Clean typography with neon accent highlights on key phrases

### 3. Skills & Expertise (4 Tabbed Sections)
- **Agentic AI & ML** — core ML capabilities listed with icon animations
- **Backend Engineering** — tech stack badges (FastAPI, Django, Spring Boot) with hover glow
- **Frontend Engineering** — React, Angular, Streamlit with visual badges
- **Cloud & DevOps** — Docker, AWS, CI/CD with infrastructure-style icons
- Each tab transitions with smooth slide/fade animations

### 4. Projects — AI Systems (Card Grid)
- 12 AI project cards in a responsive grid
- Each card has: glowing border on hover, project title, one-line description, and tech tags
- Cards animate in on scroll with staggered fade-in

### 5. Projects — Scalable Platforms (Card Grid)
- 8 non-AI project cards, same style as above
- Clear visual separation from AI section with a section header

### 6. Engineering Philosophy & Future Vision
- Stylized quote-style section with animated text reveals
- Timeline or progression visual: 2nd Year Student → AI Systems Engineer → Agentic AI Architect → Cloud-Native Platform Builder

### 7. Footer / Contact
- GitHub, LinkedIn, and Email icon links with hover glow effects
- Tagline: "Let's Build Intelligent Systems Together"
- Subtle animated background element

---

## Key Interactive Details
- Sticky navigation bar with smooth scroll to sections
- Scroll-triggered animations on every section
- Glowing neon hover effects on cards, buttons, and links
- Typing effect on hero tagline
- Staggered card reveal animations
- No backend needed — fully static, frontend-only site
